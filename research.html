Providing Haptic Feedback with Soft Robotics using Di-electric Elastomers

❮ ❯

Abstract

One of the issues users of prosthetics currently face is a lack of proprioception. Proprioception refers to ones ability to sense where their limbs are relative to their own body. One key factor in proprioception is the ability to feel, or a sense of touch. Modern prosthetics, while becoming quite adept in movement and control still lack the abilty to give the user this sense of touch, at least to a high degree. The use of soft robotics allows prosthetics to accurately simulate the shape and physical properties of tissue and skin. Soft robotics sensors can provide more accurate sensory data than rigid sensors for this reason. This project aims to use dielectric elastomers, a material used as an electromechanical transducer, as both sensors and actuators in a glove-like device providing haptic feedback to the user. I was responsible for the design of the glove device, as well as preparation of dielectric devices for characterization and implementation. I was also responsible for prototyping a 3D printed device used to characterize the force applied by the actuators.

Distinguishing Surgical Implants and Contraband in Security Contexts Using Radar

❮ ❯

Summary

While tried and tested for many years, use of metal detection in security settings is often suboptimal, with various harmless metal objects triggering false positives and causing unnecessary wasted time and distress for all those involved. This is the key motivation for this project - finding a different way to detect contraband such as weapons while reducing the number of false positives caused by things like surgical implants. This project aims to use radar data to differentiate between the two. Because this project was conducted between 2020 and 2021, mostly remotely, we used simulation software to collect data, rather than collecting data in real life simulated security settings. The software used is called SIM4Life. I was responsible for creating and automating simulations, as well as the collection and processing of data from the simulations. Because SIM4Life is an incredibly niche software, I needed to self-teach the ins and outs of the software, as well as how to integrate the data collection smoothly with Python scripts to feed into a machine learning model. Unfortunately, this project was ended before conclusive results were collected due to logistical issues with the simulation software
